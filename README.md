# JavaScript & TypeScript Roadmap

A hands-on learning repo from foundations to advanced, built for Fullstack and QA Automation engineers.

## Structure

Each section has:
- `README.md` — concepts explained + curated resources
- `exercises/` — practice problems with stubs
- `exercises/solutions/` — reference solutions

## Phases

| Phase | Topic | Level |
|---|---|---|
| [Phase 1](./phase-1-js-foundations/) | JavaScript Foundations | Beginner |
| [Phase 2](./phase-2-js-intermediate/) | Intermediate JavaScript | Intermediate |
| [Phase 3](./phase-3-ts-foundations/) | TypeScript Foundations | Intermediate |
| [Phase 4](./phase-4-ts-advanced/) | Advanced TypeScript | Advanced |
| [Phase 5](./phase-5-js-advanced/) | JavaScript Advanced (Senior) | Senior |
| [Phase 6](./phase-6-ts-advanced-2/) | TypeScript Advanced (Senior) | Senior |

## Setup

```bash
npm install
```

Run a JS file:
```bash
node path/to/file.js
```

Run a TS file:
```bash
npx tsx path/to/file.ts
```
